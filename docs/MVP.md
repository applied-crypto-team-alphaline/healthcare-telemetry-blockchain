## MVP Definition + Non-Goals

### MVP (Locked)
    Two device emulators (A and B)

    Ledger registry API: register, lookup, revoke

    Authenticated handshake (signed challenge)

    ECDH session key

    AEAD encrypted telemetry stream

    Replay detection using seq/ts

    Revoked identities rejected

    Demo script reproduces success + attack scenarios

### Non-Goals (We will NOT do)

    No full blockchain network or consensus

    No real medical devices/hardware

    No custom crypto algorithms

    No hospital/EHR integration

    No complex PKI or CA operations beyond minimal keys

    No privacy research features (DP/ZK/anonymous credentials)
